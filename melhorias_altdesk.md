# Melhorias e Sugestões - Altdesk

## Correções

### Conversas
- **Problema ao clicar em "+"**: Atualmente abre a tela de contatos, mas deveria abrir uma nova conversa apontando para um novo usuário/contato

- **Alinhamento em duas colunas**: A linha que separa o header do restante da tela deveria ser única quando há duas colunas

- **Histórico de conversas**: Uma conversa não necessariamente gera um ticket, mas deve se manter no histórico de quem abriu a conversa

- **Abertura de tickets dentro de conversas**: Adicionar funcionalidade para abrir um ticket a partir de uma conversa em andamento

- **Dificuldade em abrir conversa com outro usuário**: Definir fluxo mais claro para iniciar conversa com outro usuário

### Configuração de Empresa
- **Feriados não salvam corretamente**: Não está gravando ou não está mostrando feriado gravado

### Mensagens Rápidas
- **Edição de mensagens rápidas**: Implementar funcionalidade de editar mensagens rápidas

### Funcionalidades Gerais
- **Abrir chamado em nome de contatos/usuários**: Tanto chamados, mensagens e tickets devem poder ser abertos em nome de contatos ou usuários

- **Cabeçalho das páginas**: Resolver o cabeçalho de cada página para manter consistência com as demais páginas

## Sugestões

### Base de Conhecimento e Respostas Rápidas
- Poder usar em mensagens e tickets:
  - Base de conhecimento
  - Respostas rápidas
- Implementar botões que acessam a lista para seleção

### Arquivo de Soluções
- Permitir gravar em base de conhecimento tickets resolvidos
- Usar tickets resolvidos como referência para futuras solicitações
- Criar um repositório de conhecimento a partir do histórico de resolução
